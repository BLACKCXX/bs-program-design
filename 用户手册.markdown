# 用户手册





请按照部署指南完成部署

在项目根目录执行：

```bash
docker compose up -d --build
```

启动完成后访问：

- PC 端：`http://localhost:8080`
- 手机端（同一局域网）：`http://<电脑局域网IP>:8080`



# PC 端

## 1. 主页面

![image-20260103181922238](用户手册.assets/image-20260103181922238.png)

在该页面下， 您可以选择

**图片检索**（紫色框框出）： 您可以通过

1. 选择特定标签
2. 修改时间范围
3. 调整文件大小
4. 指定拍照设备
5. 在搜索框中输入你想要查找的相关信息

五种方法（选择单一方法或者几种方法结合）来对你想要的图片进行查找

**查看图片**（橙色框框出）： 点击图片详情页， 即可查看图片的具体信息

**上传图片**： 点击上传图片按钮， 即可实现图片的选择上传

**批量操作**： 点击批量按钮后， 便可以进行图片的批量删除、下载、添加标签操作





## 2. 图片详情页

![image-20260103182303105](用户手册.assets/image-20260103182303105.png)

点击进入图片详情页后， 您可以

- 在**基本信息栏目**中查看或修改图片的相关信息
- 在**标签栏目**中查看图片的相关标签
- 在**EXIF 信息栏**中查看图片的EXIF 信息
- 点击右上角删除按钮实现图片的删除
- 点击下载按钮讲图片下载下来
- 点击图片编辑页面开始对图片的页面编辑



## 3. 图片编辑页面

![image-20260103182902709](用户手册.assets/image-20260103182902709.png)

在图片编辑页中， 你可以：

- 对图片进行裁剪
- 对图片进行缩放操作
- 对图片进行向前90度， 向后90度的旋转
- 对图片的画面（亮度，对比度，饱和度， 色温， 锐化）等进行调节
- 对不喜欢的调节重置/回退（红框标出）
- 对图片的修改完成后导出





## 4. 搜索引擎

![image-20260103183346976](用户手册.assets/image-20260103183346976.png)

在搜索引擎中， 可以通过设置条件/ 搜索名字对想要的图片进行搜索

同时， 在下端的显示模式中， 可以点击网络，大卡片， 瀑布流等对搜索得到的图片进行设置



![image-20260103183450473](用户手册.assets/image-20260103183450473.png)





## 5. 上传中心

![image-20260103183559024](用户手册.assets/image-20260103183559024.png)

我们可以通过打开文件夹/拖动图片的方式将图片上传到上传中心当中



![image-20260103200424449](用户手册.assets/image-20260103200424449.png)

在这里我们可以

- 批量入库时点击缩略图实现对不同图片的编辑切换

- 为图片设置标题、描述、标签等

- 设置图片的可见性

- 对图片的部分信息（包括格式、大小、创建时间等）进行预览

- 利用AI 大模型自动增加标签等字段（建议使用这个， 和MCP接口更好的配合）

  









## 6. AI工作台（MCP接口）

![image-20260103183612754](用户手册.assets/image-20260103183612754.png)



- 在AI 工作台中， 我们可以与AI 聊天对话，并输入图片相关的信息对图片进行搜索
- 也可以点击AI 讲解按钮让AI 对这些图片进行概括
